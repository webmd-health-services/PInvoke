# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'PureInvoke'
Import-Module -Name 'PureInvoke'
```

To install privately:

```powershell
Save-Module -Name 'PureInvoke' -Path '.'
Import-Module -Name '.\PureInvoke'
```
